# CheburMail

**Мессенджер со сквозным шифрованием для Android, работающий поверх обычной электронной почты.**

Никаких своих серверов. Не нужен номер телефона. Сообщения идут через стандартный IMAP/SMTP (Яндекс Почта, Mail.ru), но почтовый провайдер видит только метаданные — никогда содержимое.

[English README](README.md)

## Как это работает

CheburMail превращает вашу почту в защищённый канал связи:

1. **Входите** под своей Яндекс или Mail.ru почтой (пароль приложения)
2. **Обмениваетесь ключами** — сканируете QR-код при личной встрече
3. **Сообщения шифруются** на устройстве перед отправкой — провайдер видит только набор символов
4. **Доставка через IMAP/SMTP** — без промежуточных серверов, без регистрации

```
Вы (шифрование) → SMTP → Почтовый провайдер → IMAP → Получатель (расшифровка)
                          ↑ видит только метаданные
```

## Безопасность

| Уровень | Технология |
|---------|-----------|
| Обмен ключами | X25519 (Curve25519 Diffie-Hellman) |
| Шифрование | XSalsa20-Poly1305 (аутентифицированное) |
| Хранение ключей | Android Keystore + Tink AEAD |
| Реализация | libsodium через lazysodium-android |
| Медиа | Отдельный ключ на каждый файл |

- **Zero-knowledge**: почтовый провайдер не может прочитать сообщения
- **Без своих серверов**: нечего взломать, заблокировать или запросить по решению суда
- **Уникальный nonce** на каждое сообщение
- **Верификация**: сравнение отпечатков ключей через QR-код

## Возможности

- **Личные и групповые чаты** со сквозным шифрованием
- **Медиа-сообщения**: фото, файлы, голосовые — всё зашифровано
- **Ответ, цитирование, удаление** сообщений (удаляет и из IMAP)
- **Статусы доставки** и прочтения
- **Исчезающие сообщения** с настраиваемым таймером
- **Бэкап ключей** — экспорт/импорт ключей шифрования
- **Мульти-аккаунт** — переключение между почтовыми ящиками
- **Фоновая синхронизация** через WorkManager + IMAP IDLE
- **Защита от скриншотов** на экранах чатов
- **Без трекинга, аналитики и рекламы**

## Поддерживаемые провайдеры

| Провайдер | IMAP | SMTP |
|-----------|------|------|
| Яндекс Почта | `imap.yandex.ru:993` | `smtp.yandex.ru:465` |
| Mail.ru | `imap.mail.ru:993` | `smtp.mail.ru:465` |

> Нужно создать **пароль приложения** в настройках безопасности почты. CheburMail никогда не видит ваш основной пароль.

## Сборка из исходников

**Требования**: JDK 17, Android SDK (API 35)

```bash
export ANDROID_HOME=/path/to/android-sdk
export JAVA_HOME=/path/to/jdk-17

git clone https://github.com/shizzka/cheburmail.git
cd cheburmail
./gradlew assembleDebug --no-daemon
# APK → app/build/outputs/apk/debug/app-debug.apk
```

## Стек

- **Язык**: Kotlin 2.3
- **UI**: Jetpack Compose + Material 3
- **Крипто**: lazysodium-android (libsodium) + Google Tink
- **Транспорт**: JavaMail (IMAP/SMTP)
- **Хранение**: Room + Encrypted DataStore
- **Синхронизация**: WorkManager + IMAP IDLE foreground service
- **QR**: ZXing (генерация) + Google Code Scanner (сканирование)
- **Min SDK**: 26 (Android 8.0)

## Формат сообщений

Сообщения используют свой формат, невидимый обычным почтовым клиентам:

- **Subject**: `CM/1/<chatId>/<messageUuid>`
- **Body**: `Base64(nonce || ciphertext)`
- **Content-Type**: `application/x-cheburmail`

## FAQ

**Почему email?**
Email — это федеративная, проверенная временем инфраструктура, которая уже есть везде. Не нужно поднимать серверы, нет затрат на инфраструктуру, нет единой точки отказа.

**Может ли почтовый провайдер прочитать мои сообщения?**
Нет. Сообщения шифруются на устройстве до того, как попадут на почтовый сервер. Провайдер видит отправителя, получателя и время — но содержимое неотличимо от случайных данных.

**Что будет, если я потеряю устройство?**
Используйте функцию бэкапа ключей для экспорта. Без ключей ваши сообщения не смогут быть расшифрованы — даже вами.

**Можно ли использовать Gmail / Outlook?**
Пока нет. Сейчас поддерживаются Яндекс Почта и Mail.ru. Добавить новых провайдеров несложно — PR приветствуются.
